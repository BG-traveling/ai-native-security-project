# AI 기반 사용자 이상행위 탐지 시스템
## 1. 프로젝트 소개
로그인 시도를 기록하고, 이상행위를 탐지하는 Python 기반 콘솔 프로그램입니다.
사용자의 로그인 정보를 기반으로 다음과 같은 보안 이상행위를 감지합니다:
- 일정 시간 내 과도한 로그인 시도
- 허용되지 않은 IP에서의 접근

## 2. 개발 목적
이 프로젝트는 Python 백엔드 개발자로 전환하기 위한 학습 및 포트폴리오 목적으로 개발되었습니다.

보안솔루션 엔지니어 경험을 바탕으로,  
사용자 로그인 데이터를 분석하여 이상행위를 탐지하는 시스템을 직접 구현하는 것을 목표로 합니다.

## 3. 주요 기능
- 로그인 로그 생성 및 저장
- JSON 파일 기반 로그 관리
- 과도한 로그인 시도 감지 (1분 내 4회 이상)
- 허용되지 않은 IP 접근 감지
- 사용자 입력 기반 로그 추가 기능
- 허용 IP 관리 기능
  > 추가
  > 삭제
  > 목록 조회
- IP 형식 검증
- 입력값 검증 (빈값 방지, status 제한)

## 4. 실행 방법
1. Python 실행 환경 준비
2. 프로젝트 디렉토리 이동
3. 아래 명령어 실행
``` bash
python main.py
``` 

## 5. 메뉴 설명
1. 로그 초기화
2. 테스트 로그 실행
3. 로그 출력
4. 로그 직접 입력
5. 허용 IP 추가
6. 허용 IP 삭제
7. 허용 IP 목록 보기
8. 종료

## 6. 데이터 저장 방식
- logs.json: 로그인 로그 저장
- allowed_ips.json: 허용된 IP 목록 저장

## 7. 기술 스택
- Python
