# AI 기반 사용자 이상행위 탐지 시스템
## 1. 프로젝트 소개
AI를 활용하여 사용자의 행위에 대한 감시 및 로깅을 진행하는 시스템입니다.

## 2. 개발 목적
해당 시스템을 개발하게 된 이유는 보안솔루션 엔지니어라는 직무를 수행하면서 
이런 부분은 이렇게 처리됐으면 좋겠다라는 아이디어를 통해서 해당 시스템을 개발하게 되었습니다.
현재 아직 개발 진행 상태는 아니며, 먼저 시스템 설계 및 어떠한 도구를 사용하여 개발을 할지 검토중입니다.

## 3. 주요 기능 (예정)
1. 사용자 로그인 로그 기록
2. 과도한 요청 감지
