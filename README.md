# AI 기반 사용자 이상행위 탐지 시스템
## 1. 프로젝트 소개
사용자의 로그인 행위를 기반으로 이상행위를 탐지하는 시스템입니다.
로그 데이터를 수집하고, 특정 기준에 따라 비정상적인 접근을 감지합니다.

## 2. 개발 목적
보안솔루션 엔지니어로 근무하면서
사용자 접근 로그를 분석하는 과정에서
이상행위를 자동으로 탐지할 수 있는 시스템의 필요성을 느껴 개발하게 되었습니다.

## 3. 주요 기능 (예정)
1. 사용자 로그인 로그 생성 및 저장
2. 일정 시간 내 과도한 로그인 시도 감지
3. 허용되지 않은 IP 접근 감지
4. 로그 출력 기능 (단일 / 전체)

## 4. 기술 스택
- Python

## 5. 현재 진행 상태
- 로그인 로그 생성 기능 구현 완료
- 로그 리스트 저장 구조 구현 완료
- 과도한 로그인 요청 감지 기능 구현 완료
- 허용되지 않은 IP 접근 감지 기능 구현 완료
- 로그 출력 함수 분리 완료

## 6. 현재 파일 구조
- log_manager.py
 -> 로그 생성, 저장, 탐지, 출력 관련 함수 모듈

- main.py
 -> 테스트 실행 코드
